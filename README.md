# Firefox Installer Scripts for Linux

Easily install **Firefox Stable, Developer Edition, or Nightly** on Linux with a single command.  
No root privileges required — installs to your home directory and creates a desktop entry automatically.

---

## Install Firefox Stable Edition ![Firefox](https://searchfox.org/firefox-main/source/browser/branding/official/default24.png)

Run the following command in your terminal:

```bash
curl -s https://raw.githubusercontent.com/tazihad/firefox-installer/refs/heads/main/firefox-stable-installer.sh | bash
```
