# Firefox Installer Scripts for Linux

Easily install **Firefox Stable, Developer Edition, or Nightly** on Linux with a single command.  
No root privileges required â€” installs to your home directory and creates a desktop entry automatically.

---

## Install Firefox Stable Edition ![Firefox](https://upload.wikimedia.org/wikipedia/commons/thumb/a/a0/Firefox_logo%2C_2019.svg/24px-Firefox_logo%2C_2019.svg.png)

Run the following command in your terminal:

```bash
curl -s https://raw.githubusercontent.com/tazihad/firefox-installer/refs/heads/main/firefox-stable-installer.sh | bash
```
